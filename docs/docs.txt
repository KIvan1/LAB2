Модуль show
Метод show

Тест suite1
Цель: проверить корректность работы функции при расположении курсора в последней строке
Тип: позитивный
Входные данные: text txt
Ожидаемый результат: в стандартный поток вывода выведенно содержимое структуры text строки разделены '\n' и в конце - символ курсора '|'
Описание процесса: перехватываем поток вывода записываем в строку output и сверяем эту строку со строками структуры text. Сравнение идет посимвольно, и строки структуры text не содержат в конце '\n', поэтому после прохождения по очередной строке text (кроме последней) пропускаем один символ.

Описание процесса: перемещаем курсор в позицию 1 2. Перехватываем поток вывода записываем в строку output и сверяем эту строку со строками структуры text. Сравнение идет посимвольно, и строки структуры text не содержат в конце '\n', поэтому после прохождения по очередной строке text пропускаем один символ. В строке 1 в позиции 2 пропускаем один символ т. к. это курсор.
